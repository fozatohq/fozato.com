# Fozato | AI-Powered YouTube SEO SaaS

**Fozato** is a modern SaaS platform offering AI-powered YouTube SEO automation. Built with **TailwindCSS** and **React** / **Next.js**, Fozato enables content creators, businesses, and marketers to automate SEO tasks, boost video rankings, and grow their channels effortlessly. Join a global network of creators who trust Fozato to optimize their content. Created with passion for creators by our dedicated team.

## Usage

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

### Getting Started

1. Clone the repository.
2. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
